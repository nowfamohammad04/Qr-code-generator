## 📌 Overview

This is a modern **QR Code Generator and Scanner Web Application** built using HTML, CSS, and JavaScript.

It allows users to:

* Generate QR codes from text, URLs, or images
* Scan QR codes from uploaded images
* Download generated QR codes
* View decoded content instantly

---

## ✨ Features

* 🔳 Generate QR codes (text, URL, image)
* 📷 Scan QR codes from uploaded images
* 📂 Drag & Drop image upload
* 🔗 Clickable links from decoded QR
* 🖼 Image preview from QR data
* ⬇ Download QR code as PNG
* 📋 Auto copy QR data to clipboard
* 🔔 Toast notifications
* 📱 Fully responsive modern UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* QRious (QR generation)
* jsQR (QR decoding)
